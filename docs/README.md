# Solafon API Documentation

Welcome to the Solafon API documentation. Solafon is a platform that enables developers to create and deploy mini-applications that users can interact with through a chat-based interface.

## What is Solafon?

Solafon provides:
- **Mini-Apps Platform** - Create interactive apps that users can discover and use
- **Chat Interface** - Users interact with your apps via messages and commands
- **Developer Tools** - Full API for sending/receiving messages, webhooks, and more
- **Built-in Moderation** - All apps go through moderation before going live
- **Mana Points System** - Internal currency for platform features

## Key Features

### For Users
- Browse apps by categories (AI, Games, Trading, DeFi, NFT, and more)
- Interact with apps through familiar chat interface
- Use Mana Points for premium features
- Secret Login for enhanced privacy

### For Developers
- Create apps via Dev Studio or API
- Set up webhooks to receive user messages
- Define custom commands with auto-responses
- Access comprehensive Developer API
